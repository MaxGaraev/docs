---
title: "Authentication"
description: "Аутентификация запроса к API"
---

Чтобы получить доступ к частному агенту или хранилищу данных, вы должны предоставить ключ API DataUp в заголовке авторизации вашего запроса.

```bash
'Authorization': 'Bearer <API_KEY>'
```

## API Keys
​
Ключ API вы можете найти на странице настроек профиля.
